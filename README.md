# Graph Network Optimization: Minimum Spanning Tree

## О проекте

Реализация алгоритма построения минимального остовного дерева (Minimum Spanning Tree) для оптимизации городской инфраструктурной сети.

Задача проекта — соединить здания города сетью с минимальной суммарной длиной кабеля.

Для построения MST используется алгоритм Краскала и собственная реализация структуры Disjoint Set Union (DSU).

---

## Реализовано

* структура непересекающихся множеств (DSU);
* алгоритм Краскала;
* построение графа городской инфраструктуры;
* работа с географическими координатами;
* визуализация полученной сети на карте.

---

## Алгоритмы и структуры данных

### Disjoint Set Union

Реализованы операции:

* MakeSet;
* FindSet;
* Union.

Для оптимизации использованы:

* path compression;
* union by rank.

Это позволяет выполнять операции практически за константное время.

---

### Алгоритм Краскала

Этапы работы:

1. сортировка рёбер по весу;
2. последовательное добавление рёбер;
3. проверка циклов через DSU;
4. построение минимального остовного дерева.

---

## Данные

Использованы реальные географические данные зданий города.

Каждая вершина графа соответствует зданию с координатами:

* latitude;
* longitude.

Рёбра графа отражают возможные соединения между ближайшими зданиями.
